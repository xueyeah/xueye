Sep 08, 2019Linux点击

WebSSH

​ WebSSh github 地址参考： https://github.com/huashengdun/webssh

​ webssh是在浏览器上已一个网页的形式模拟客户端crt等操作linux机器的一个工具。安装该工具预置条件为

Python 2.7/3.4+

测试机器Python 版本信息：

注：在python 2.7 版本下需要安装 setuptools 与 pip （python3.4+ 不需要安装setuptools）

安装setuptools

切换到root 用户下

1.下载 setuptools-41.0.1.zip 上传至/home/nms/ 目录解压

下载地址：https://pypi.org/project/setuptools/#files

进入 /home/nms/ setuptools-41.0.1 目录

安装pip

  1. 下载 pip-19.1.1.tar.gz 上传至 /home/nms/ 目录 解压（下载地址 https://pypi.org/project/pip/#files）

进入 /home/nms/pip-19.1.1 目录 执行

  1. 安装 webssh
  1. 开启webssh服务

webssh服务 127.0.0.1:8888

  1. 浏览器输入如下地址即可模拟web端登陆linux服务器

    注：192.168.23.146 为安装webssh服务机器地址

    ​ 135.251.208.160 为登陆机器目标ip(安装webssh服务机器要与目标机器互通)

    ​ username 目标机器登陆用户名

    ​ password目标机器登陆用户密码
